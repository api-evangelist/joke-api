# JokeAPI

JokeAPI is a free, open-source REST API that delivers consistently formatted jokes with no authentication, registration, or payment required. It provides over 1,368 jokes across categories including Programming, Misc, Dark, Pun, Spooky, and Christmas in 6 languages.

**Human URL:** https://jokeapi.dev/
**Base URL:** https://v2.jokeapi.dev

## Categories

- Any
- Programming
- Misc
- Dark
- Pun
- Spooky
- Christmas

## Content Flags

- NSFW
- Religious
- Political
- Racist
- Sexist
- Explicit

## Response Formats

- JSON
- XML
- YAML
- Plain text

## Rate Limits

- 120 requests per minute per client (IP-based)
- 5 joke submissions per minute

## Authentication

No API key or registration required.
